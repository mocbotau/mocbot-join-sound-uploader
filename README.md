# MOCBOT Join Sound Uploader

This is a website used to interface Discord users with MOCBOT v2's join sound feature.

## Setup

### Required Environment Variables

- `VITE_AUTH0_DOMAIN` - The auth0 application domain
- `VITE_AUTH0_CLIENT_ID` - The auth0 application client ID
- `VITE_AUTH0_CALLBACK_URL` - The callback URL to be used by auth0
- `VITE_AUDIENCE_URL` - The audience URL for the auth0 application
- `VITE_API_URL` - The base URL for the API
- `VITE_MOC_GUILD_ID` - The ID of the Discord guild
